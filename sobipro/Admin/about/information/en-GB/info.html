You are using the trial version of SobiPro

The purpose of the trial version is to show you how SobiPro looks like and what it can, but has a limited feature set. With a subscription, you get the full version of SobiPro.

The full version gives you access to the following additional features:

  • Adding more sections
    The trial version is limited to one section only.
  • Installing applications manually
    The trial version does not allow installing applications manually.
    Applications can be installed only via the repository.
  • Choosing other templates than the default one.
    The trial version is limited to the default template.
    Please note, the default template is subject to change on update of SobiPro!
  • Adding more than 20 fields
    The trial version is limited to 20 fields.
  • Crawling the site to build the SobiPro cache in a fast way
    In the trial version, the crawler is disabled.
  • Possibility to change the payment trial data to real one
    In the trial version, the demo data for PayPal and Offline payments cannot be changed.
  • Action Log: Log the modifications made
    In the trial version you cannot log the modifications made in your SobiPro installation.
  • Versioning of entries
    In the trial version, versioning of entries is not available. You cannot go back to a previous version of an entry.

With a SobiPro subscription, additionally, you will get access to ...

  • additional applications for SobiPro
  • the full documentation for SobiPro and its applications
  • professional support in our ticket system.
